# Estrategia de Contenido — Instagram & Facebook
### DisruptLab · Ejecutable por 1 persona con IA

*Versión 1.0 — Marzo 2026*

---

## PRINCIPIO BASE

No tenemos equipo. Tenemos IA.
El contenido tiene que ser **constante, específico y creíble** — no perfecto.
Un post mediocre publicado hoy gana a un post perfecto que nunca sale.

**Regla de oro:** El 90% del contenido da valor. El 10% vende.

---

## LOS 5 PILARES DE CONTENIDO

Cada post pertenece a uno de estos 5 pilares. Rotar siempre entre todos.

| Pilar | % del contenido | Objetivo | Ejemplo |
|-------|----------------|----------|---------|
| **1. Educación** | 30% | Autoridad + búsqueda | "3 errores que hacen que tu Instagram no venda" |
| **2. Prueba social** | 25% | Confianza + conversión | Antes/después de web de cliente |
| **3. Relatable** | 20% | Alcance + compartidos | "Cuando el cliente dice que lo hace su sobrino 🙄" |
| **4. Behind the scenes** | 15% | Humanización | "Así armamos un bot de WhatsApp en 48hs" |
| **5. Venta directa** | 10% | Leads | "2 lugares disponibles este mes — auditoría gratuita" |

## FORMATOS POR PLATAFORMA

### Instagram
| Formato | Alcance | Esfuerzo | Cuándo usarlo |
|---------|---------|----------|---------------|
| **Reel (30–60s)** | 🔥 Máximo | Medio | Educación + relatable + lanzamientos |
| **Carrusel (5–10 slides)** | Alto | Medio | Educación detallada + tutoriales |
| **Post imagen única** | Medio | Bajo | Frases + datos + prueba social |
| **Story** | Alto (base de seguidores) | Muy bajo | Diario — detrás de escenas, encuestas, CTAs |

### Facebook
| Formato | Cuándo usarlo |
|---------|---------------|
| **Post de texto largo** | Historias personales + reflexiones del negocio |
| **Reel compartido desde IG** | Mismos reels de Instagram, subidos directo a FB también |
| **Post imagen + copy** | Tips rápidos + casos de éxito |
| **Lives** | Auditorías públicas en vivo (mes 2+) |

## WORKFLOWS CON IA — SIN EQUIPO DE CONTENIDO

### Flujo completo para un post (20–45 min)

```
1. IDEA (2 min)
   → Elegir del banco de ideas o de lo que pasó esta semana

2. COPY (5 min)
   → Prompt para Claude/ChatGPT (ver sección de prompts)

3. IMAGEN (10 min)
   → Prompt para Canva AI / FLUX / Leonardo (ver sección de prompts)

4. EDICIÓN (5 min)
   → Ajustar texto en Canva, agregar logo, paleta de colores

5. PUBLICAR (3 min)
   → Instagram + compartir a Facebook
```

## PROMPTS DE COPY PARA CLAUDE/CHATGPT

### Prompt base para cualquier post educativo:

```
Sos el founder de DisruptLab, una agencia digital para PYMEs en Argentina.
Escribí un post de Instagram sobre: [TEMA]
Target: dueños de negocios en Buenos Aires (estéticas, psicólogos, gimnasios, dentistas, inmobiliarias)
Tono: directo, cercano, sin tecnicismos. Como si hablaras con un conocido.
Formato: Hook impactante en la primera línea + 3-5 puntos concretos + CTA al final
Largo: máximo 150 palabras
NO usar emojis en exceso. Máximo 3 por post.
NO usar palabras como "sinergia", "ecosistema", "soluciones integrales"
```

### Prompt para post relatable/meme:

```
Sos el founder de DisruptLab, agencia digital para PYMEs en Argentina.
Creá un post de Instagram con humor que resuene con dueños de negocio que tienen problemas con su presencia digital.
Situación a retratar: [SITUACIÓN — ej: "el dueño que responde WhatsApps a las 2am"]
Formato: texto corto + punch final
Tono: humor inteligente, no burlarse del cliente sino empatizar
```

### Prompt para caption de antes/después:

```
Escribí el caption para un post de Instagram que muestra el antes/después de [TIPO DE NEGOCIO].
Antes: [describir estado anterior — web inexistente, Instagram sin estrategia, etc.]
Después: [describir resultado — web nueva, bot activo, etc.]
Incluir: el dolor que tenía, la solución que aplicamos, el resultado en números si hay
CTA final: invitar a agendar auditoría gratuita
Largo: 100 palabras máximo
```

### Prompt para Story con encuesta:

```
Creá una pregunta para Story de Instagram que genere respuesta y me ayude a entender qué necesitan los dueños de negocio.
Tema: [TEMA — presencia digital, WhatsApp, redes sociales, etc.]
Formato: pregunta + 2 opciones de respuesta (para encuesta de IG)
Debe ser genuinamente interesante para el target
```

```
Minimalist dark background, professional design for a digital marketing agency post.

Color palette: dark background #07080a, accent emerald green #10b981, white text.
No people. Abstract digital elements. Professional and elegant.
Aspect ratio: 1:1 for Instagram feed.

```
Split screen comparison image. Left side: outdated, messy business presence (old website, chaotic WhatsApp, no branding). Right side: modern, clean, professional digital presence (sleek website, automated chat, cohesive brand).
Style: flat design illustration, dark background, green accent colors.
Labels: "ANTES" and "DESPUÉS" in bold white text.
```

### Para posts relacionables/meme:

```
Simple, bold meme-style image. Clean background, large text overlay.
Style: minimal, modern, not stock photo. Could be illustrated character or pure typography.
Mood: [HUMOR / RELATABLE / EMPÁTICO según el post]
Brand colors: dark background, emerald green accent.
```

```

## ESTRUCTURA DE UN REEL EDUCATIVO (sin cámara)

Para el founder que no quiere salir en cámara todavía:

```
[0–2s] HOOK en texto grande:
"El error que le cuesta clientes al 80% de los negocios en Instagram"

[2–8s] EL PROBLEMA (texto + imagen):
Mostrar screenshot de un negocio con solo Instagram y WhatsApp
Texto: "Solo tenés Instagram → el cliente llega a las 11pm → nadie responde → se va"

[8–20s] LA SOLUCIÓN (3 puntos rápidos):
✅ Web con formulario → captura el lead dormido
✅ Bot de WhatsApp → responde automático
✅ Google Business → te encuentran antes de Instagram

[20–28s] PRUEBA:
Screenshot de resultado o demo

[28–30s] CTA:
"Auditoría gratuita → link en bio"
```

**Producción:** Canva animado → exportar → CapCut para subtítulos + música → publicar
